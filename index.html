Sweepline Voronoi Diagram (Fortune's Algorithm)

This demonstrator shows how Fortune's Algorithm cane be used to construct a Voronoi diagram from a set of points. To start the demonstration, either add points to the plane below and press "Start Algorithm", or try one of the demo diagrams with the buttons above.

Your browser doesn't support canvas. This demonstrator can therefore not be displayed.
This is an open source demonstrator, built by Christopher Oser. The implementation is largely based on the source code of Raymond Hill https://github.com/gorhill/Javascript-Voronoi. Many thanks and praises go out to Mr. Hill.
